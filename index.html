PROGRAMA EDUCATIVO

Ingeniería en Tecnologías de la Información forma profesionales capaces de diseñar, implementar y gestionar soluciones tecnológicas innovadoras que respondan a las necesidades del entorno empresarial y social.

¿Conoces la carrera de Ingeniería en Tecnologías de la Información e Innovación Digital?

ASIGNATURA

Propósito: El alumno implementará sistemas empleando tecnologías web para extender la disponibilidad y alcances de los servicios de la organización.

La asignatura de Programación Web se cursa en el séptimo cuatrimestre de la carrera de Ingeniería en Tecnologías de la Información.

UNIDADES DE APRENDIZAJE

Unidad I: Introducción a los sistemas web

Propósito: Configurar entornos de trabajo para el desarrollo web.

Temas
  • Introducción a la programación web
  • Configuración de servidores de desarrollo web
  • Configuración de servidores de producción
Proceso de evaluación

    Reporte digital sobre cómo instalar servidores web

  • Evidencia: El reporte debe incluir los pasos, capturas de pantalla y una lista del software usado.
  • Lugar: Práctica en el laboratorio, usando una computadora.
  • Evaluación: Caso práctico y rúbrica.

Unidad II: Programación del lado del cliente

Propósito: Elaborar documentos responsivos y dinámicos del lado del cliente para el despliegue del contenido del sistema web.

Temas
  • Maquetación semántica
  • Hojas de estilo con documentos HTML responsivos
  • JavaScript en documentos HTML dinámicos
Proceso de evaluación

    Reporte digital sobre la elaboración de un sistema web

  • Evidencia: El reporte debe incluir el código HTML, la hoja de estilos (CSS), los archivos del proyecto y la descripción del proceso.
  • Lugar: Práctica en el laboratorio, usando una computadora.
  • Evaluación: Caso práctico y rúbrica.

Unidad III: Programación del lado del servidor

Propósito: realizará la programación y monitoreo de aplicaciones en servidores web para complementar la funcionalidad del sistema web.

Temas
  • Introducción a la programación del lado del servidor
  • Interfaz de programación de aplicaciones y programación de tres niveles
  • Acceso a servicios
  • Monitoreo y estadísticas del sistema web
Proceso de evaluación

    Reporte digital a partir de un caso práctico de programación del lado del servidor

  • Evidencia: El reporte debe incluir el código fuente, la documentación, capturas de pantalla de la conexión segura y los registros de monitoreo.
  • Lugar: Se realizará como una práctica en el taller, usando una computadora.
  • Evaluación: Caso práctico y rúbrica.

Unidad IV: Autenticación de usuarios

Propósito: Implementará mecanismos de autenticación de usuarios para el acceso al sistema web.

Temas
  • Autenticación e interfaces seguras
  • Manejo de excepciones
Proceso de evaluación

    Reporte digital a partir de un caso práctico de autenticación de usuarios en el servidor de aplicaciones web

  • Evidencia: El reporte debe incluir la descripción del proceso para implementar seguridad y el código fuente del manejo de excepciones.
  • Lugar: Se realizará como una práctica en el taller, usando una computadora.
  • Evaluación: Caso práctico y rúbrica.
